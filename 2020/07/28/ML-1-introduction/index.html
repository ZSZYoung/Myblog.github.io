    Lecture1-introduciton

    What is ML

    Learning Map

    一些概念：

    Regression

    即回歸，指找到一個func通過input特征得到一個數值型的輸出。

    Classification

    即分類，指找到一個func通過input特征得到一個分類型的輸出。

    Binary Classification

    結果僅有兩類的分類

    Multi-class Classification

    結果有多類的分類

    學習方法

    Supervised Learning

    尋找function時，讓機器通過input和對應output(label) 尋找function。這些label往往是人工標註的。

    Reinforcement Learning

    尋找function時機器不知道對於input的正確答案是什麼，只得到一個reward，再由reward情況機器進行自我引導學習。

    Aloha Go是先由棋譜進行監督式學習再通過強化學習得到的。

    Meta Learning

    讓機器學習如何得到使機器學習的function。
